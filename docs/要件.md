# ライフプランシミュレーター 要件定義・仕様書（確定版）

## 1. プロジェクト概要
本プロジェクトでは、「賃貸」「新築戸建」「中古戸建」「新築マンション」「中古マンション」の5つの住宅プランを軸に、将来の収支・資産推移を詳細にシミュレーション・比較できるウェブアプリケーションを構築しました。
公的統計や最新の税制（2025年の住宅ローン減税等）に基づいた精緻な計算ロジックを備え、ユーザーが自身のライフイベントや経済状況に合わせた最適な住居選択を行えるよう支援することを目的としています。

## 2. システム要件（実装済み機能）

### 2.1 機能要件
- **複数プラン比較機能**: 
    - 5種類の住宅パターン（賃貸、新築/中古戸建、新築/中古マンション）をベースに複数プランを作成し、同一グラフ上で比較。
    - 各プランごとに個別のローン条件、修繕費、管理費などを設定可能。
    - スライダーを用いた直感的なパラメータ操作とリアルタイムなグラフ描画。
- **生涯収支シミュレーション**:
    - 現在の年齢から死亡想定年齢（デフォルト90歳）までの、年次ベースでの資産推移と収支の計算。
    - 建物の経年劣化（減価償却）を考慮した「不動産価値」と、金融資産からローン残債を引いた「純資産」の可視化。
- **ライフイベント設定**:
    - **共通ライフプラン**: すべてのシミュレーションに横断的に適用されるイベント（例：子育て費用のプリセット、車の購入など）。
    - **個別ライフプラン**: 特定のプランにのみ適用されるイベント（例：持ち家のリフォーム費用、賃貸の更新・引越し費用など）。
- **マクロ経済シナリオ**:
    - インフレ率、運用利回りの設定によるキャッシュフロー変動の反映。
- **ビジュアライゼーション**:
    - 純資産推移の複数ライン折れ線グラフ。
    - プラン単体の収支内訳（収入、生活費、住居費、ローン控除など）を示す積み上げ面グラフ。
- **データ管理と出力**:
    - Zustand と `localStorage` を用いた入力条件のブラウザへの自動保存・復元。
    - 全プランの年次キャッシュフロー詳細と資産推移データを、AI解析に適したロングフォーマット（Tidy Data）のCSVとしてダウンロードする機能。

### 2.2 非機能要件
- **プラットフォーム**: 最新のモダンブラウザ（Chrome, Edge, Safari）。
- **レスポンシブデザイン**: Tailwind CSSによる適応的なレイアウト（主にPC/タブレット利用を想定した高密度ダッシュボード設計）。
- **デプロイ・ホスティング**: GitHub Pages への静的エクスポートによる完全なクライアントサイド稼働（サーバーサイドでの個人情報保持なし）。

## 4. 計算ロジックの仕様

### 4.1 住宅ローンと控除（2025年版対応）
- **毎月の返済額**: 元利均等返済方式で算出。
- **住宅ローン控除**:
    - その年の年末ローン残高の **0.7%** を算出。
    - 入力された「物件種別・環境性能」および「世帯属性」に基づく **借入限度額** を上限として適用。
    - 控除期間は新築が **13年間**、中古が **10年間**。
    - （※本シミュレーターでは、簡易的に設定された年収から控除可能な十分な税額がある前提で全額を還元として計算）

### 4.2 資産価値の減価償却
- 総資産 ＝ 金融資産 ＋ （土地価値 ＋ 建物残存価値）
- 建物残存価値は、設定された「建物の耐用年数」に基づく直線法で毎年償却され、耐用年数経過後は0（土地価値のみ残存）として計算されます。

### 4.3 キャッシュフローの推移
1. **年次収入**: 給与成長率を加味して手取り額を概算増加。定年以降は収入0（または年金想定額）で計算。
2. **年次支出**: インフレ率を加味した基本生活費 ＋ 住宅関連費（ローン返済・維持費・家賃等） ＋ ライフイベント支出（共通・個別）。
3. **運用益**: 年始の金融資産残高に対して、設定された「運用利回り」を乗じて算出。
4. **残高更新**: 次年度の初期金融資産 ＝ 前年度残高 ＋ (年次収入 － 年次支出) ＋ 運用益 ＋ ローン控除還付金。

## 5. 技術・アーキテクチャスタック
- **Frontend**: Next.js 15 (App Router, Exportモード) / React 19 / TypeScript
- **Styling**: Tailwind CSS v4
- **State Management**: Zustand (persist ミドルウェアによるローカルストレージ永続化)
- **Charts**: Recharts (レスポンシブなSVGベースのチャート描画)
- **Icons**: Lucide React
- **CI/CD**: GitHub Actionsを用いたGitHub Pagesへの自動デプロイ

## 6. 将来の拡張構想（未実装要件）
本アプリのインフラおよびコンポーネント設計は、以下の高度な機能追加に耐えうるよう構築されています。
- **確率的シミュレーション（モンテカルロ法）**: 利回りやインフレ率などの変数に分散を持たせ、1000回以上の試行によって「90歳時点での破産確率」などを算出する機能。
- **高度な税金・年金計算**: 額面年収からの精緻な所得税・住民税控除モジュールや、「ねんきん定期便」のデータを用いたリタイア後の収入シミュレーションの統合。
- **データのインポート/クラウド同期**: JSONファイルによるプラン設定の共有機能やクラウドへのデータバックアップ。
