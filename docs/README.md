# ライフプランニング・シミュレーター (Life Plan Simulator)

2025年の最新の住宅ローン減税制度や物価上昇リスクに対応した、高度なライフプラン・シミュレーションWebアプリケーションです。賃貸、新築戸建て、中古マンションなど、複数の住居形態を比較し、将来の資産推移を可視化します。

## 主な機能

1.  **複数プランの比較シミュレーション**
    *   新築マンション、中古マンション、新築戸建て、中古戸建て、賃貸の5種類のベースプランから複数を作成し、同時に比較が可能。
    *   年齢ごとに純資産（金融資産 ＋ 不動産価値 － ローン残債）の推移を折れ線グラフで可視化。

2.  **精緻な住宅コスト計算**
    *   **2025年以降の住宅ローン減税**に対応（ZEH水準、長期優良住宅、子育て世帯の上乗せ措置などを網羅）。
    *   マンションの「修繕積立金の段階的増額（5年ごとにX%上昇など）」をシミュレーション可能。
    *   中古物件の仲介手数料や、新築マンションの修繕積立基金など、初期費用を物件種別ごとに自動算出。
    *   建物の減価償却に基づく将来の売却想定価値（出口戦略）を計算。

3.  **ライフイベントの管理**
    *   全プラン共通の「共通ライフプラン」（車の購入、子育て・教育費用のセット入力など）を一括反映。
    *   各プラン個別の「個別ライフプラン」（例：持ち家のリフォーム費用、賃貸の引越し費用）をそれぞれ設定可能。

4.  **高度な経済シナリオ設定**
    *   給与上昇率、インフレ率、資産運用利回りを個別に設定し、将来のキャッシュフローに反映。

5.  **優れたユーザー体験 (UX) とデータ保存**
    *   リアルタイムでグラフが更新されるインタラクティブなUI。
    *   ブラウザの `localStorage` を使用した入力データの自動保存（リロードしてもデータが消えません）。
    *   シミュレーション結果の年次キャッシュフロー表をAI解析用のロングフォーマットCSVとして出力可能。

## 技術スタック

*   **フレームワーク**: Next.js 15 (App Router, React 19)
*   **スタイリング**: Tailwind CSS v4
*   **状態管理**: Zustand (persist ミドルウェアによるローカルストレージ永続化)
*   **グラフ描画**: Recharts
*   **アイコン**: Lucide React
*   **デプロイ**: GitHub Pages (Static HTML Export)

## 開発環境のセットアップ

```bash
# クローンと依存パッケージのインストール
npm install

# 開発サーバーの起動 (http://localhost:3000 にアクセス)
npm run dev

# プロダクションビルド
npm run build
```

## ディレクトリ構成
- `src/app/` : Next.js App Routerのエントリーポイント (`page.tsx`, `layout.tsx`)
- `src/components/` : UIコンポーネント
  - `PlanDetailEditor.tsx` : 各プランの詳細設定モーダル
  - `Sidebar.tsx` : 共通プロファイル・ライフイベント設定
  - `charts/SimChart.tsx` : 資産推移の折れ線グラフ
  - `charts/StackedSimChart.tsx` : 収支内訳の積み上げグラフ
- `src/store/` : Zustandを用いた状態管理 (`usePlanStore.ts`)
- `src/lib/` : ユーティリティとシミュレーションエンジン
  - `engine.ts` : 資産推移を計算するコアロジック
  - `types.ts` : TypeScriptの型定義
- `docs/` : ドキュメント類

## デプロイメント

このプロジェクトは GitHub Pages への静的エクスポート (`output: 'export'`) に対応しています。
`.github/workflows/deploy.yml` により、`main` ブランチへのプッシュ時に自動的にビルドとデプロイが行われます。

*   ローカル開発時は `http://localhost:3000/` で動作します。
*   本番環境（GitHub Pages）では `next.config.ts` により自動的に `basePath: '/lifeplan-simulator'` が付与されます。
